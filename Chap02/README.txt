table1.xlsx
Manually curated info extracted from literature on selected pathogenicity genes for A. fumigatus

protein.fasta
Sequences for all the A. fumigatus genes in table1 was downloaded from UNIPROT (stored in Protein_fasta) and blasted against the reference genome for Aspergillus species

combine_command.sh
combines all single fasta contained in protein_fasta into the single file combined.fasta

b_load_clean.R 
Function clean out of blast and add coloumn names
